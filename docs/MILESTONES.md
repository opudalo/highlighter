# Product milestones

## Hackathon MVP — implemented

- Rights-safe prepared library and local-only Neuromancer flow.
- Shared EPUB 2/3 parser, fingerprint matching, stable blocks, and IndexedDB persistence.
- Responsive chapter reader with visible-paragraph tracking and exact-position name actions.
- Spoiler-safe dossier, local evidence, timelines, aliases, and deterministic one-hop graph.
- Resumable forward-only preprocessing with Codex, Claude fallback, checkpoints, overrides, validation, and ignored QA reports.
- Parser, boundary, pipeline, import, persistence, and interaction tests.
- GitHub Pages build with relative assets and an explicit two-EPUB production allowlist.

## After the hackathon

- Add an artifact review interface for identity merges and low-confidence records.
- Split large processed artifacts into lazy-loaded book/chapter bundles.
- Add reader typography controls and richer EPUB navigation semantics.
- Expand accessibility testing with multiple screen readers and mobile browsers.
- Add more legally distributable prepared editions.
