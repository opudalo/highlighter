# Demo EPUB assets

- `alice.epub` — *Alice’s Adventures in Wonderland*, Lewis Carroll, Standard Ebooks CC0 edition.
- `frankenstein.epub` — *Frankenstein*, Mary Shelley, Standard Ebooks CC0 edition.
- `neuromancer.epub` — local-development-only user copy; explicitly ignored and excluded from production builds.

Standard Ebooks dedicates the entirety of its produced ebook files, including markup and cover art, to the public domain through the CC0 1.0 Universal Public Domain Dedication:
